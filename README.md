# Linear Algebra Library in C

Учебный проект по дисциплине «Низкоуровневое программирование».

## Описание проекта

Проект представляет собой библиотеку линейной алгебры на языке C для работы с векторами, матрицами и решения систем линейных алгебраических уравнений (СЛАУ).

Библиотека реализована в модульной форме и включает:
- операции с векторами;
- операции с матрицами;
- решение СЛАУ методом Гаусса;
- тесты для проверки корректности работы функций;
- демонстрационную программу.
